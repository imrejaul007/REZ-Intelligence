# REZ Approval UI

Human-in-the-loop approval dashboard for AI-drafted campaigns and actions.

## Features

- **Pending Approvals Queue** - View all items awaiting approval
- **Quick Actions** - Approve, Reject, or Escalate with one click
- **Content Preview** - See exactly what AI drafted
- **Priority Levels** - Urgent, High, Medium, Low
- **Status Tracking** - Know what's approved, rejected, or escalated
- **Notes** - Add context to decisions

## Item Types

- **Campaign** - Marketing campaigns to launch
- **Budget** - Budget reallocation requests
- **Message** - Auto-generated customer messages
- **Action** - System actions (pause, resume, etc.)
