# REZ Visual Workflow Builder UI

Drag-and-drop visual workflow builder for campaign automation.

## Features

- **Drag-and-Drop Canvas** - Visual workflow editing
- **Node Palette** - Pre-built nodes for triggers, actions, logic, and flow control
- **Node Configuration** - Easy configuration panel for each node
- **Workflow Templates** - Start from pre-built templates
- **Execution Dashboard** - Monitor workflow runs

## Node Types

### Triggers
- Schedule (cron-based)
- Event (order, signup, cart, etc.)
- Webhook
- Manual

### Actions
- Send WhatsApp/SMS/Email/Push
- Create Campaign
- Update CRM
- Add Loyalty Points

### Logic
- If/Else conditions
- Filter audience
- A/B Split testing

### Flow
- Delay/Wait
- For Each loop
- End

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```
